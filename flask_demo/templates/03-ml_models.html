Text Clustering

Clustering is an unsupervised machine learning method that converts unstructured data into structured data to get some sense out of it. This app uses natural language processing (NLP) techniques to clean up free texts and k-means to cluster them.

Inputs
This model needs number of clusters and a text file with extension .csv or .txt containing at least a column of text like below. English-only please.

doc_id text
1 I'm happy today, can't you see it shows on my face? Says a stoic.
2 Sadness is my new normal. Welcome depression! Or where are you sunlight?
3 I am experiencing mania. Such a spiritual bliss.

Output
A zipped excel file with sheets comprised of cluster assignment, top keywords for each cluster, and number of documents in each of the clusters respectively.

API
You can also access the model through the Flasgger API.

  • Expand the /text_clustering_api
  • Press button "Try it out"
  • Upload your input file and define the desired number of clusters
  • Press button "Execute"
  • In the Responses section, find Response body and click "Download file"
